## Compilação


Digite no terminal para compilar com cmake:
```console
mkdir build
cd build
cmake ..
cmake --build .
```


## Execução

Para executar o programa, dentro da pasta build digite o seguinte comando:
```console
./snake PATH
```

Sendo PATH o caminho até os arquivos que contem os labirintos.
